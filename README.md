# 💳 FamPay Gateway

## 🚀 Features
- ✅ Web interface se Gmail App Password dalna
- ✅ Real-time payment monitoring
- ✅ QR code generation
- ✅ Create payment orders
- ✅ Automatic balance updates

## 🔧 Setup
1. Open `/setup` page
2. Enter:
   - Gmail Email
   - Gmail App Password (16-digit)
   - FamPay UPI ID
3. Click "Start Monitoring"

## 🚀 Deploy to Vercel
```bash
vercel --prod
